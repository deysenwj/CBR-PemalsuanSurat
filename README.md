# CBR-Pemalsuan-Surat

Case-Based Reasoning (CBR) System for Document Forgery Detection

## Project Structure

```
CBR-Pemalsuan-Surat/
├── data/
│   ├── raw_pdf/      # Original PDF documents
│   ├── raw_text/     # Extracted text from PDFs
│   ├── processed/    # Preprocessed data
│   ├── eval/         # Evaluation datasets
│   └── results/      # Experiment results
├── notebooks/         # Jupyter notebooks for exploration
├── models/           # Saved models and case bases
└── scripts/          # Utility scripts
```

## Requirements

- Python 3.11

## Installation

```bash
pip install -r requirements.txt
```
